# 商业模式诊断报告

**日期：** 2026-04-06
**诊断工具：** dbs-diagnosis (体检模式)
**诊断对象：** Dash — Custom Digital Systems

## 基本信息
- **业务：** 用 AI agent 编排（harness engineering）为酒店运营方构建定制化软件系统
- **产品：** HCO（Hotel Cashback Operations）— 酒店返现数据对账 + 工作流管理系统
- **价格：** $4,800 / $7,500 / $10,800 三档项目制（已选 Option II: $7,500）
- **付款节奏：** 50% 签约 → 30% 验收 → 20% 上线
- **月收入：** $0（第一单审核中）
- **客户来源：** 庞帅介绍 → Capricorn（实际运营方）主动对接需求

## 诊断结果

### 1. 印钞机检验：部分通过

**Input：** 你的时间 + Claude Code/OpenClaw ($200/月) + 酒店 domain knowledge + 审美判断力
**Output：** 定制化酒店运营系统

机器能跑，但依赖你这个人。产品有价值，但交付方式不可复制。你造了一台只有你会开的车。

**壁垒分析：** 你的壁垒不在技术（AI 编排可被学习），在于「懂酒店业务 + 会用 AI 工具 + 有审美」的组合稀缺性。但这个稀缺性是暂时的。

### 2. 道德检验：通过

好模式。产品让客户更透明（不可篡改审计日志）、更高效（替代 Excel 手动对账）。你不需要夸大或制造焦虑来成交。赚钱的方式不需要做坏事。

### 3. 定价检验：需要调整

$7,500 项目制可以用于第一单验证，但不是长期模式。

**问题：**
- 三档价格倍数不够（I→II 1.56x，II→III 1.44x），更像同一产品的三个配置
- 项目制 = 一次性收入，收入上限 = 能接几个单 × $7,500
- 年收入天花板约 $15,000-22,500

**处方：** 第一单用项目制交付（验证产品）。第二单起转订阅：$200-500/月/酒店。项目费变成获客成本——它帮你打磨了产品。

### 4. 需求检验：通过（强信号）

客户原话痛点（来自 Capricorn 3/18-3/22 聊天记录）：
1. 收集到的姓名和供应商反馈的姓名有差异（张三丰 → Zhang Sanfeng / Sanfeng Zhang / Mr. Zhang...）
2. 部分酒店将一个预定拆成多个单日预定，共享确认号
3. 确认号会复用，不能依靠确认号检索订单
4. 酒店名称在客户端、酒店实际、供应商返回三方之间有差异

**核心需求：** 三方数据对不上号。不是「要一个网站」，是「不用再每天手动对 Excel」的解脱。这是行业通用痛点，不是一家独有。

### 5. 流量-变现关系检验：需要建设

获客渠道为零。第一个客户来自人脉介绍。第二个客户的来源完全未知。

**核心动作：** 交付后从 Capricorn/庞帅处获取同行情报——「你知道还有谁在用 Excel 对 cashback 账吗？」这一句话的价值超过任何营销策略。

### 6. 规模化检验：底层可规模化，交付方式不可

HCO 的核心逻辑（模糊匹配 + 状态机 + 审计日志 + RLS）是行业通用的。第一单是打磨产品，第二单起必须是「开账号」不是「做项目」。

**关键发现：** 用户确认是 Option A（复制 + 改配置 + 部署新实例），不是 Option B（从头构建）。这意味着离 SaaS 只差一个多租户功能。

### 7. 成长层级：第 1→2 层过渡中

| 层级 | 描述 | 状态 |
|------|------|------|
| 1 | 有人需要这个产品 | **通过** — Capricorn 主动找你，痛点真实 |
| 2 | 有人愿意付钱 | **审核中** — 合同 $7,500 已发，等签字 |
| 3 | 有很多人愿意付钱 | 未验证 |
| 4-7 | 太远 | 不要想 |

第 2 层的标志：$3,750（50% 首付）到银行账户。

## 核心判断

最大资产不是 DashPersona，不是奖项，不是 harness engineering 的概念——是 **HCO 解决了一个真实的、行业通用的、客户愿意出 $7,500 的痛点**。这在所有项目里是唯一一个有市场拉力信号的。

最大风险是**分心**。DashPersona、网易云音乐、Zonic、LinkedIn 个人品牌——这些都在消耗注意力，但没有一个在产生收入。

## 一句话处方

**你不是缺方向，你是缺签收的那一笔钱。停止思考「以后的路怎么走」，去把合同签了。**
